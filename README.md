# CapsNet_pytorch
- Refer to [naturomics/CapsNet-Tensorflow](https://github.com/naturomics/CapsNet-Tensorflow)(Tensorflow) and [timomernick/pytorch-capsule](https://github.com/timomernick/pytorch-capsule)(Pytorch).
